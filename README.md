
# GoBarber - back end

Sistema de CRUD seguindo o modelo API REST para ser consumido futuramente pelo GoBarber - front end.

## Principais Comandos

### Instalar dependencias do projeto
```
:~$ cd <pasta do projeto>
:~$ yarn
ou
:~$ npm install
```
### Como excutar o projeto?
```
:~$ yarn start
ou
:~$ npm start
```
